# 🛍 Retail Sales & Customer Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end Business Intelligence and Data Analytics solution built to analyze retail sales data and generate actionable business insights.

The dashboard provides executive-level KPIs, customer segmentation, sales forecasting, and interactive data exploration to support data-driven decision making.

The project is implemented using both:

• 🐍 Python (Streamlit + SQL + Machine Learning)
• 📊 Power BI (Interactive Business Dashboard)

---

## 🎯 Business Objectives

- Analyze total revenue and profit performance
- Identify top-performing products and categories
- Segment customers using RFM analysis
- Monitor monthly growth trends
- Forecast future sales using Machine Learning
- Build an executive-style interactive dashboard

---

## 🚀 Features

### 📊 Executive KPI Dashboard
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %
- Average Order Value
- Monthly Growth %

### 📈 Sales Analysis
- Sales by Category
- Sales by Segment
- Monthly Sales Trend
- Top 10 Products

### 👥 Customer Analytics
- RFM-based Customer Segmentation
- Cluster Visualization

### 🔮 Forecasting
- Sales Forecast using Linear Regression
- Historical vs Predicted Trend Comparison

## ▶ How to Run (Python Version)

1. Clone the repository:

   git clone <your_repo_link>

2. Navigate to project folder:

   cd Retail_Analytics_Dashboard

3. Install dependencies:

   pip install -r requirements.txt

4. Run the dashboard:

   streamlit run app.py

5. Open in browser:

   http://localhost:8501
